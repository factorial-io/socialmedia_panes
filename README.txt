Social Media Panes
===========

Integrates with various social media platform like facebook and twitter.
Exposes a custom pane that can be added to a panel page.

Facebook Integration

1) Create a facebook Page
  https://www.facebook.com/pages/create.php
2) Create a facebook App for the page
  https://developers.facebook.com/apps
3) Configure Social Feed Settings
  http://www.example.com/admin/config/services/socialfeed
4) Template for a single row for facebook feed can be found in
  <path to this module>/templates/socialmedia_panes_facebook.tpl.php
5) Template for the entire wrapper can be found in
  <path to this module>/templates/socialmedia_panes_social_media.tpl.php

--------------------------------------------------------------------------------

Twitter Integration

1) Create a Twitter account
  https://twitter.com/signup
2) Create a Twitter App for the account
  https://apps.twitter.com/
3) Configure Social Feed Settings
  http://www.example.com/admin/config/services/socialfeed/twitter
4) Template for a single row for twitter feed can be found in
  <path to this module>/templates/socialmedia_panes_twitter.tpl.php
5) Template for the entire wrapper can be found in
  <path to this module>/templates/socialmedia_panes_social_media.tpl.php
